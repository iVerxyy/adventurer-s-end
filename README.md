# Adventurer's End
Navigate treacherous dungeons filled with mysterious black holes that pull you closer every 2 seconds. Push boxes onto goals while managing your precious rewind energy - because in this adventure, mistakes cost more than time.

## Origin

Originally, Adventurer's end was a game, made for the Codédex 2025 Game Jam, it was a mini-game project, made with CSS, HTML, and JavaScript
